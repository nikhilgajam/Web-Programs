Transforms 2D

2D Transform Options. To know more click here

Options

  • translate

    • transform:translate(x,y);
      move x pixels to left(-)/right(+) and y pixel up(-)/down(+)
      transform:translate(100, 75);

  • rotate

    • transform:rotate(xdeg);
      Rotate/spin the element a certain number of degrees
      transform:rotate(60deg);

  • scale

    • transform:scale(width, height);
      Change the width and height of the element
      transform:scale(2, 3);

  • skew

    • transform:skew(x-angle, y-angle);
      Rotate the element a certain number of degrees along the x and y axis
      transform:skew(40deg, 15deg);

  • matrix

    • transform:matrix() - combines all of the 2D transform methods into one
      transform:skew(40deg, 15deg);

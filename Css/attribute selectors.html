attribute selectors

we can style like this a[href="info.html"]{code}

we can do that also

Operators

^ : match the beginning exactly
Ex: a[href^ = "http://umich"]

$ : match the end exactly
Ex: img[src$ = ".png"] => this is for .png images

* : wild
Ex: a[href* = "umich"]

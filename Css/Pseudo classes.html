Pseudo-Classes

Types of Pseudo-Classes:

Link

  :link, :visited

User Action

  :hover, :active, :focus

forms(interfaces)

  :enabled, :checked, :disabled


Note: These actions will work on all of the tags not only links

Structural/Positional

  :first-child, :last-child, :nth-child(), :only-child
  :first-of-type, :last-of-type, :only-of-type

Ex:

li:first-child{ code }
li:nth-child(3){ }
p:empty{ }
img:only-of-type{ }
p:last-of-type{ }


Textual

  :first-letter, :first-line


Positional/Generated

  :before, :after


fragments

  ::selection
